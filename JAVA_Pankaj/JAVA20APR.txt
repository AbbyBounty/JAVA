Typecasting:
		    1) Auto Upcasting
			->Converting smaller datatype to bigger datatype
		    2) Explicit Down Casting

->moment of data from smaller to bigger is "Upcasting"
->moment of data from bigger to smaller is "Down-casting"

Automated smaller to bigger.
Explicitly bigger to smaller because can't done automatically .

long 8 byte
double 8 byte
boolean size is not applicable

int i=10;
long j=i;  //upcasting from 4 bytes to 8 bytes
